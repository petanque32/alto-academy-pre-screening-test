# Flowchart Diagram

```vbnet
Start
Input number 1
Input operator
Input number 2
If operator is +
    Add number 1 and number 2
Else If operator is -
    Subtract number 2 from number 1
Else If operator is *
    Multiply number 1 and number 2
Else If operator is /
    If number 2 is 0
        Display error message "Division by zero"
    Else
        Divide number 1 by number 2
Else
    Display error message "Invalid operator"
End

```
